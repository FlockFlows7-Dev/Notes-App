# Notes App

A simple C# console app to create, view, and clear notes.

## How to Run

1. Clone this repository to your local machine.
2. Open the solution in Visual Studio (or your preferred C# IDE).
3. Run the program.

## What It Does

- Create new notes
- View all saved notes
- Clear all notes
